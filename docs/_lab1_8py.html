Detailed Description

This file contains code which initializes a motors encoder connected to the designated pins using the encoder_reader class.

Then it initializes the motor driver using the motor_driver class. After both are initialized, the program gives the motor a set of duty cycles to run for 2 seconds each. Finally, the encoder is called to read position every .5 seconds.
